# Portfolio

Personal portfolio built with Next.js 15, React 19, TypeScript, Tailwind CSS, and deployed on AWS through Serverless Framework.

Live site: [thomasott.fr](https://thomasott.fr)

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Biome
- Yarn 4 PnP
- AWS S3 + CloudFront

## Development

```bash
yarn install
yarn dev
```

## Quality checks

```bash
yarn lint
yarn format:check
yarn build
```

## Deployment

```bash
yarn deploy:dev
yarn deploy
```
